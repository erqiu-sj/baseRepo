# 打包规范

- `__test__`
  
用于存放测试文件

- `dts`
  
用于存放 `package` 声明文件

- `dist`
  
存放 `esm` 规范的 `package` 产物,配置类 `package` 忽略

- `lib`
  
存放 `commonjs` 规范的 `package` 产物
